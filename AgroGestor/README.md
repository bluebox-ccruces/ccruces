# AgroGestor

Aplicativo móvil Android (maqueta) para gestión operativa agrícola.

## Tecnologías
- React Native (Expo)
- Diseño: Industrial Clean & High Contrast

## Pantallas
- Login
- Dashboard
- Registro Operativo
- Resumen y Modificación
- Información y Soporte

## Colores
- Primario: Azul Petróleo (#1A5276)
- Fondo: Gris Humo Claro (#F8FAFB)
- Acción: Naranja Seguridad (#F39C12)
- Éxito: Verde Esmeralda (#27AE60)

## Tipografía
- Inter Sans

## Notas
- Solo maqueta visual, sin lógica de backend ni sincronización real.
- Para Android, pero puede verse en web con Expo.

---

### Comandos útiles

```sh
cd AgroGestor
npm start # Expo Go
npm run android # Emulador Android
```
